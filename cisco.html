Hálózatszimuláció és gyakorlás

📚 Modulok

1
Alap
▼

Alapvető parancsok - Belépés konfigurációs módba: enable → configure terminal - Kilépés konfigurációs módból: end vagy Ctrl+Z - Konfiguráció mentése: write memory vagy copy running-config startup-config Interfészek beállítása 1. Válaszd ki az interfészt: interface GigabitEthernet0/0 2. Adj IP-címet: ip address Példa: ip address 192.168.1.1 255.255.255.0 3. Kapcsold be az interfészt: no shutdown DHCP konfiguráció 1. DHCP pool létrehozása: ip dhcp pool 2. Hálózat megadása: network Példa: network 192.168.1.0 255.255.255.0 3. Átjáró megadása: default-router 4. DNS szerver megadása: dns-server 5. Kiosztott címek megtekintése: show ip dhcp binding Statikus routing Statikus útvonal beállítása: ip route Példa: ip route 192.168.2.0 255.255.255.0 10.0.0.2

Hostnév és MOTD Banner hostname R1 banner motd #Illetéktelen belépés tilos!# 3. Ellenőrzés Futó konfiguráció megtekintése: show running-config Aktív VTY kapcsolatok ellenőrzése: show users

3
OSPF
▼

Belépés OSPF konfigurációs módba: enable configure terminal router ospf A egy lokális azonosító a routeren belül, például 1. Hálózatok hozzáadása OSPF-hez: network area : az OSPF által kezelni kívánt hálózat címe : fordított alhálózati maszk (pl. 255.255.255.0 → 0.0.0.255) : az OSPF terület azonosítója, általában 0 az alapértelmezett backbone area Példa: network 192.168.1.0 0.0.0.255 area 0 network 10.0.0.0 0.0.0.255 area 0 Ellenőrző parancsok OSPF-hez: show ip ospf neighbor # OSPF szomszédok ellenőrzése show ip route ospf # OSPF által tanult útvonalak show ip ospf interface # OSPF interfész státusz

4
VLAN
▼

Hálózati Parancs Segédlet 1. VLAN-ok létrehozása és kezelése (switch) • vlan Új VLAN létrehozása. • name VLAN elnevezése. 2. Access portok beállítása (PC-k felé) • interface fastethernet0/x Port kiválasztása konfiguráláshoz. • switchport mode access A port access módban működjön. • switchport access vlan Port hozzárendelése egy VLAN-hoz. 3. Trunk portok beállítása • switchport mode trunk A port trunk módban vigye több VLAN forgalmát. • switchport trunk allowed vlan Engedélyezett VLAN-ok megadása trunkon. 4. Router-on-a-stick – alinterfészek • interface gigabitethernet0/x Fizikai interfész kiválasztása. • no shutdown Interfész engedélyezése. • interface gigabitethernet0/x. Alinterfész létrehozása VLAN-hoz. • encapsulation dot1Q 802.1Q VLAN tag megadása. • ip address IP-cím beállítása alinterfészen. 5. Interfész a DHCP szerver felé • interface gigabitethernet0/x Másik hálózathoz csatlakozó interfész. • ip address IP beállítása DHCP kapcsolat számára. • no shutdown Interfész aktiválása. 6. DHCP relay (ip helper-address) • ip helper-address DHCP kérések továbbítása másik hálózat felé. 7. DHCP szerver routeren • ip dhcp excluded-address Kiosztásból kizárt címek. • ip dhcp pool DHCP tartomány létrehozása. • network Hálózat megadása DHCP-nek. • default-router Gateway cím megadása kliensek számára. • dns-server DNS szerver beállítása. • show ip dhcp binding
