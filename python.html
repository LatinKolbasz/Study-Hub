Class (Osztályok)
▼

1. Class létrehozása A class egy objektumtervet (blueprint) ad, amiből példányokat (objektumokat) tudsz csinálni. class Auto: def __init__(self, marka, szin): self.marka = marka self.szin = szin def inditas(self): print(f"A {self.marka} autó elindult!") # Példányosítás auto1 = Auto("Toyota", "piros") auto2 = Auto("BMW", "kék") # Metódus hívása auto1.inditas() # A Toyota autó elindult! auto2.inditas() # A BMW autó elindult! __init__ → konstruktor, amikor létrejön az objektum self → az aktuális objektumra utal self.marka és self.szin → objektum változói (attribútumok) inditas() → metódus (függvény a class-on belül) 2. Attribútumok módosítása print(auto1.szin) # piros auto1.szin = "fekete" # módosítás print(auto1.szin) # fekete 3. Több metódus class Auto: def __init__(self, marka, szin): self.marka = marka self.szin = szin def inditas(self): print(f"A {self.marka} autó elindult!") def festes(self, uj_szin): self.szin = uj_szin print(f"A {self.marka} autót újrafestették {uj_szin}-re") auto = Auto("Honda", "fehér") auto.inditas() # A Honda autó elindult! auto.festes("fekete") # A Honda autót újrafestették fekete-re 4. Összefoglalva Class → tervrajz Objektum → class példánya Attribútum → objektumhoz tartozó adat Metódus → objektumhoz tartozó függvény __init__ → konstruktor, a létrehozáskor fut le self → mindig az adott objektumra utal

Loopok (Ciklusok)
▼

1. if feltétel x = 7 if x > 5: print("x nagyobb, mint 5") elif x == 5: print("x pont 5") else: print("x kisebb, mint 5") if → igaz feltétel esetén lefut elif → másik feltétel, ha az if nem teljesült else → minden más eset 2. while ciklus count = 0 while count < 5: print("Szám:", count) count +=1 # mindig növelni kell, különben végtelen Addig fut, amíg a feltétel igaz. 3. for i in range ciklus for i in range(5): print("i=", i) range(5) → 0, 1, 2, 3, 4 range(start, stop, step) → például range(2, 10, 2) eredménye: 2,4,6,8 If + for kombináció for i in range(1, 6): if i % 2 == 0: print(i, " páros") else: print(i, "páratlan" )
